# Historical Software

This is the list of recovered software, ready to run on emulator test-run .zip files.
Just download the selected test-run, unzip it, and execute the appropriate .bat file
All source code taken from original listings, except when noted.
<br>
Software available to play with:
<br>

- 1951    Ferranti Mark 1 Scheme A 
- 1952    Ferranti Mark 1 Scheme B 
- 1952    Ferranti Mark 1 Glennie Autocode (rebuild)
- 1952    Ferranti Mark 1 Love Letters 
- 1952    Ferranti Mark 1 Draugths game 
- 1952    IBM 701 NR9003 Assembly (rebuild)
- 1952    IBM 701 SO2 Regional Assembler (rebuild)
- 1953    Ferranti Mark 1 Scheme C 
- 1953    IBM 701 SpeedEx North American Aviation Assembler 
- 1954    Ferranti Mark 1 Brooker Autocode 
- 1954    Ferranti Mark 1 Scheme T 
- 1954	IBM 650 Interpretive FDS 
- 1955	IBM 650 Regional Assembler 
- 1955    IBM 701 PACT I compiler (rebuild)
- 1956	IBM 650 Bell Interpretive System 
- 1956	IBM 650 IT 
- 1957	IBM 650 SOAP II 
- 1957	IBM 650 FORTRANSIT 
- 1958    IBM NORC PERT 
- 1959	IBM 650 SOAP IIA-4000 Build tape Library 
- 1959	IBM 650 SuperSoap
